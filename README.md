RH Insight IA : Analyseur de Marque Employeur

    Automatisez votre benchmark RH et décryptez l'expérience collaborateur grâce à l'IA.

Ce projet est une application de business intelligence dédiée aux Ressources Humaines. Elle permet de comparer visuellement les entreprises et d'extraire une synthèse des avis salariés en utilisant des modèles de langage.

Points Forts du Projet

    Analyse Sémantique Haute Performance : Utilisation de l'API Groq avec le modèle Llama 3.1 (8B) pour une analyse quasi-instantanée.

    Interface Décisionnelle : Dashboard interactif avec indicateurs colorés dynamiques basés sur l'Index Égalité Professionnelle et la satisfaction salariés.

    Architecture "Production-Ready" :

        Caching : Optimisation des performances et réduction des coûts API via @st.cache_data et @st.cache_resource.

        Robustesse : Parsing JSON structuré pour garantir la fiabilité des extractions de l'IA.

        Sécurité : Gestion rigoureuse des secrets et des erreurs système (Fail-safe).

🛠️ Stack Technique

    Langage : Python 3.9+

    Framework Web : Streamlit

    IA / LLM : Groq API (Llama 3.1)

    Analyse de données : Pandas

    Formatage : JSON structuré
